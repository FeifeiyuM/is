is [![Build Status](https://travis-ci.org/issue9/is.svg?branch=master)](https://travis-ci.org/issue9/is)
======

一些常用的验证函数：
```go
// 判断是否为数值
is.Number("123")

// 判断是否为ISBN序列号
is.ISBN("1-919876-03-0")
```

### 安装

```shell
go get github.com/issue9/is
```
